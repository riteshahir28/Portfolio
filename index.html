Blog Website Details

This is a blog-based web application where users can create their own profile and share blog posts (books, articles, thoughts, etc.) with others. After creating an account, users can log in and access their dashboard to post blogs, edit or delete them, and manage their content.

Other users can view posted blogs, like them, and read full content. Each blog post shows the number of likes and views it has received. There’s also a rating feature to allow feedback on posts. All users have their own profile where they can see their posted blogs.

An admin panel is available to manage user activities and moderate content. The UI is mobile-responsive and user-friendly.

Key Features:

  • User registration and login system
  • Create and post blog articles (books, thoughts, etc.)
  • Like, view count, and rating system on posts
  • User profile page showing personal posts
  • Admin panel to manage users and content
  • Responsive design using HTML, CSS, Bootstrap

Technologies Used: Java, JSP, Servlets, MySQL, Bootstrap, JavaScript, HTML, CSS

Library Management System Details

This is an online examination system where students can register, log in, and appear for exams based on their selected course or subject. The application is designed to make exam management easy for both students and the admin.

Once logged in, students can view available exams, start any exam, and submit their answers within a time limit. After submission, the system automatically checks the answers, calculates the score, and shows the result to the student.

The admin panel allows admin users to add new questions, create exams, manage users, and view all results. The system stores all data securely in the database and ensures that no exam can be attempted twice by the same user.

Key Features:

  • User (student) registration and login
  • Subject-wise online exams
  • Timer-based automatic exam submission
  • Instant result display with scoring
  • Admin panel to manage exams, questions, and students
  • One exam per student per subject (no repeat)
  • Fully responsive UI

Technologies Used: Java, JSP, Servlets, MySQL, HTML, CSS, Bootstrap, JavaScript
